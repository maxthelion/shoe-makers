# Add known-issue entry for npm registry 403 errors in cloud environments

skill-type: doc-sync

## What to add

Add a new section to `.shoe-makers/known-issues.md` documenting the npm registry 403 error pattern:

### npm registry returns 403 in cloud environments

**Symptom**: `bun install` fails with 403 errors on all packages from registry.npmjs.org.

**Cause**: Some cloud environments block or restrict npm registry access. The package registry may not be available.

**Fix**: If bun is already installed globally (`which bun`), the system can still work — `bun run setup` and `bun test` use local files that don't require npm. Skip `bun install` and proceed. If dependencies are missing, the setup script will fail with import errors — at that point, the elf should report the issue and stop.
