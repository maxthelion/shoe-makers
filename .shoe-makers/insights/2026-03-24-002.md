# Insight: Mycorrhizal Network — Expand known-issues.md into skill-specific knowledge

## Lens: *Mycorrhizal Networks from Forest Ecology* — Underground fungal networks connect trees, allowing them to share nutrients, water, and chemical signals.

## Connection

`.shoe-makers/known-issues.md` already serves as shared knowledge between elves, but it only covers troubleshooting. This shift revealed several "gotchas" that would help future elves but aren't bugs:

- Evidence patterns don't match markdown bold (`**text**`) in template literals
- Claim IDs are auto-generated by slugifying to 60 chars — must verify exact IDs
- The `###` regex only matches two-level numbering (2.6), not three-level (2.6.1)

These are facts that reduce reactive churn if known in advance.

## Proposal (reworked)

Expand `known-issues.md` into a broader knowledge base. Add a new section "## Patterns and Gotchas" for non-bug knowledge. Include relevant entries in skill prompts (e.g., the evidence pattern gotchas would be included in explore and doc-sync prompts).

No new directories needed — just expand the existing mechanism.

## Why

- Low cost: add sections to an existing file
- Direct impact: this shift's reactive churn (3 evidence pattern mismatches) would have been avoided if the elf knew the gotchas
- Builds on existing infrastructure rather than creating new conventions
